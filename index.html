Quick reference for using the Trigger Simulator

Overview

Use the Home view to build trigger systems visually. Use the toolbar(on the left) to add triggers, pulse/send channels, and save/load layouts.

Adding a Trigger

To add a trigger, fill out the form in the "Add Trigger" section of the toolbar. The fields are as follows:

  • ID: Unique identifier for the trigger (e.g., T_A, T_B)
  • Delay: Time in seconds before the trigger activates after receiving the activate signal.
  • Activate On: Channel name that activates the trigger when pulsed. (Can be a list, "a, b, c".)
  • Deactivate On: Channel name that deactivates the trigger when pulsed. (Can be a list, "a, b, c".)
  • Trigger On: Channel name that causes the trigger to send its output when pulsed (if active). (Can be a list, "a, b, c".)
  • When Triggered: Channel name that this trigger will pulse when it successfully triggers. (Can be a list, "a, b, c".)
